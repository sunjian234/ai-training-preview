# 版本记录

本文件记录“AI职业技能培训与智能体应用项目官网”的重要变更。

## V0.1.0 校长预览版 — 2026-07-21

### 新增

- 建立完全独立的原生静态网站项目。
- 完成首页首屏、项目介绍、学校合作价值、三级课程体系、实训项目、证书与能力证明、就业方向、企业合作、Hermes 概念性实训平台、首期试点班、合作联系和页脚合规声明。
- 建立学校、学员、企业三类对象的内容表达。
- 展示“招生获客 → 技能培训 → 实操考核 → 能力证明 → 人才就业 → 企业项目服务 → 持续运营”价值闭环。
- 增加移动端导航、页面内平滑跳转、导航状态提示和返回顶部功能。
- 增加不发送、不存储数据的预览版合作意向表单。
- 增加响应式布局、键盘焦点、跳过导航、语义标签和减少动态效果支持。
- 增加 GitHub Pages 部署说明、合规报告、移动端检查、功能测试、文件清单和本地预览说明。

### 合规边界

- 明确本页面为项目规划展示版，不构成正式招生、收费、证书发放或就业承诺。
- 区分学校培训合格证明、项目实训能力证明和国家职业技能衔接方向。
- Hermes 仅作概念性功能介绍，不包含内部资料。
