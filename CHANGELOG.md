# 版本记录

本文件记录“AI职业技能培训与智能体应用项目官网”的重要变更。

## V0.2.0 多页面内容版 — 2026-07-21

### 升级

- 将单页网站升级为多页面结构。
- 新增九个一级栏目。
- 新增下拉子目录和手机折叠菜单。
- 完善课程、证书、就业、学校合作、企业服务和 Hermes 平台内容。
- 增加面包屑、二级导航和相关页面推荐。
- 保留项目规划展示和合规边界。

### 技术与体验

- 建立首页与八个一级内页，统一使用公共样式和原生 JavaScript。
- PC 端提供固定主导航、当前页高亮和悬停/键盘下拉菜单。
- 手机端提供汉堡菜单、子目录展开、遮罩关闭和 Esc 关闭能力。
- 所有资源和页面链接使用适配 GitHub Pages 子目录的相对路径。
- 关于我们页面保留不联网、不存储的合作意向表单。

## V0.1.0 校长预览版 — 2026-07-21
