# 本地预览与截图说明

## 推荐预览方法

在项目目录中启动本地静态服务器：

```powershell
python -m http.server 8080 --bind 127.0.0.1
```

浏览器打开：

```text
http://127.0.0.1:8080/
```

结束预览：在运行服务器的终端按 `Ctrl + C`。

## 直接打开

也可双击项目根目录下的 `index.html`。本项目使用相对资源路径，不依赖后端或构建工具。

## 建议截图清单

本版本未自动保存截图文件。需要制作校长演示材料时，建议按以下尺寸截图：

- 桌面首页首屏：1440 × 900。
- 桌面学校合作价值：1440 × 900。
- 桌面课程与实训项目：1440 × 1000。
- 手机首页：390 × 844。
- 手机课程与合作表单：390 × 844。

截图前请确认浏览器缩放为 100%，页面加载完成，且画面中不出现浏览器账号、收藏夹、通知或其他私人信息。

## 重点人工浏览路径

1. 首屏阅读定位和五项核心能力。
2. 点击“查看学校合作”，检查校长价值内容。
3. 点击“了解学习路径”，检查三级课程。
4. 浏览九个实训项目和三层能力证明。
5. 检查就业方向没有承诺性表述。
6. 检查企业合作流程与平台流程图。
7. 在表单中分别测试空提交与填写后提交。
8. 缩小到手机宽度，打开菜单并逐项跳转。
9. 到页脚确认合规声明和版本号。
