# 本地预览与截图说明

## 推荐预览方法

在项目目录启动静态服务器：

```powershell
python -m http.server 8080 --bind 127.0.0.1
```

浏览器访问：<http://127.0.0.1:8080/>。结束预览时在终端按 `Ctrl + C`。

## 重点浏览路径

1. 首页确认项目标识、三类用户入口和概览链接。
2. 依次访问项目介绍、学习中心、证书与评价、就业与发展。
3. 从学校合作访问企业服务和 Hermes 实训平台。
4. 在关于我们页测试表单空提交与完整提交提示。
5. 在 PC 端检查悬停下拉、当前页高亮和二级锚点。
6. 在手机端检查汉堡菜单、子目录展开、遮罩关闭和横向溢出。
7. 到每页页脚确认版本号和合规边界。

## 测试与截图尺寸

- 手机：390 × 844
- 平板：768 × 1024
- 常规桌面：1366 × 768
- 大屏桌面：1920 × 1080

建议至少保存首页的 390 × 844 和 1366 × 768 截图。截图前确认浏览器缩放为 100%，页面加载完成，且不显示账号、收藏夹、通知或其他私人信息。

当前项目不依赖外部图片，即使 `assets/images/` 中没有展示素材，也能完整呈现全部页面。
