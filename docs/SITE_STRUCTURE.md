# 网站结构说明

版本：V0.2.0 多页面内容版

## 一级信息架构

网站采用“首页分流 + 八个一级内页”的结构，共 9 个 HTML 页面。

| 一级栏目 | 路径 | 主要内容 |
|---|---|---|
| 首页 | `index.html` | 项目定位、三类用户入口、价值链、课程与就业概览、合作入口 |
| 项目介绍 | `project/index.html` | 项目背景、市场需求、培养目标、项目优势、发展规划 |
| 学习中心 | `learning/index.html` | 三级课程、学习方式、九类实训项目、考核方式 |
| 证书与评价 | `certificate/index.html` | 培训证明、项目档案、国家职业衔接、条件说明、合规与问答 |
| 就业与发展 | `career/index.html` | 就业岗位、创业兼职、作品集、企业需求、服务流程、人才库 |
| 学校合作 | `school/index.html` | 合作价值、收益模式、课程共建、基地、师资、招生、流程 |
| 企业服务 | `enterprise/index.html` | 企业培训、知识库、智能体、流程优化、定向培养、交付维护 |
| Hermes实训平台 | `hermes/index.html` | 平台定位、教学与实训流程、项目管理、风险审计、作品管理、规划 |
| 关于我们 | `about/index.html` | 愿景、团队、伙伴规划、合作表单、完整合规声明 |

## 主导航与子目录

- PC：顶部固定导航；悬停、聚焦或点击一级栏目可查看下拉子目录；当前页高亮。
- 手机和平板：汉堡菜单打开侧边导航；点击一级栏目展开子目录；点击遮罩、关闭按钮或按 Esc 收起。
- 首页使用根目录相对路径，内页使用 `../` 返回站点根目录，适配 GitHub Pages 仓库子目录部署。

各内页顶部包含面包屑、页面标题、定位说明和粘性二级锚点导航。页面底部提供相关栏目推荐和返回顶部按钮。

## 首页职责

首页不重复展示所有详细正文，只承担：

1. 说明项目定位与合规性质。
2. 将学员、学校和企业分流至对应页面。
3. 概览价值链、三级课程、就业方向、Hermes平台和典型项目。
4. 引导进入学校、企业、师资及项目合作路径。

## 公共资源

- `assets/css/style.css`：统一色彩、排版、卡片、流程、导航、表单和响应式布局。
- `assets/js/main.js`：公共页眉页脚、当前页状态、下拉菜单、移动菜单、锚点状态、返回顶部和本地表单反馈。
- `assets/images/`、`assets/icons/`：为后续经授权的图片和图标预留；当前页面不依赖外部图片。

## 视觉与交互系统

- 深海军蓝与青绿色构成正式、现代的主视觉；暖金色仅用于审慎提示。
- 内页使用统一标题层级、内容卡片、流程步骤和低调醒目的合规提示框。
- 关键交互均具备悬停、键盘焦点和按下反馈。
- 支持 `prefers-reduced-motion`，不使用自动播放视频、背景音乐或强制动效。

## 数据边界

- 无后端、数据库、Cookie、浏览器持久化或第三方接口。
- 关于我们表单没有 `action`，提交由本地脚本阻止并显示预览提示。
- 不加载统计、支付、广告、外部字体或需要密钥的 API。
- 不包含账号凭据、原始 PDF、真实报名数据或内部平台资料。
