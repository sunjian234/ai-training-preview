## 表单安全验证

- 表单没有 `action`，未指定第三方目标。
- 提交事件执行 `preventDefault()` 与 `stopPropagation()`。
- 空提交会显示“请先填写姓名、联系方式和合作方向”。
- 完整提交只显示指定预览提示，不跳转、不上传。
- JavaScript 中不存在 `fetch`、`XMLHttpRequest`、`sendBeacon`、Cookie、`localStorage` 或 `sessionStorage`。
- 页面不含身份证、银行卡、密码或文件上传字段。

结论：合作意向表单不会向网络发送或持久化保存填写内容。
