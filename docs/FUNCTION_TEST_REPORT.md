## 表单安全验证

合作意向表单：

- 没有 `action` 地址。
- 提交事件执行 `preventDefault()` 和 `stopPropagation()`。
- JavaScript 不调用 `fetch`、`XMLHttpRequest`、`sendBeacon` 或第三方SDK。
- 不写入 Cookie、localStorage、sessionStorage 或 IndexedDB。
- 页面提示明确告知“不发送、不保存”。

结论：预览版表单不会把填写内容发送到网络或持久化保存。

## 按钮与反馈

- 三个首屏按钮均跳转到明确栏目。
- 顶部“合作联系”跳转到表单。
- 移动菜单按钮通过形态变化显示展开/关闭状态。
- 表单提交按钮在空字段与填写完整时显示不同的可见反馈。
- 返回顶部按钮跳转至页面顶部。
- 所有按钮和链接具备悬停、键盘焦点和按下视觉状态。

## GitHub Pages 发布检查

- HTTPS 已启用。
- GitHub Pages 发布源为 `main` 分支根目录。
- 网站首页、CSS 和 JavaScript 使用适配仓库子路径的相对地址。
- 页面源码使用 UTF-8，并保留简体中文内容。
- 表单仍无提交目标，不调用网络接口或浏览器持久化存储。

正式对外推广前，仍建议在 Chrome、Edge、Safari、Firefox 最新稳定版及真实手机上复核菜单、软键盘和长页面滚动体验。
